把搜索栏做出来了，目前功能只是预设关键字跳转到指定链接。

## 带参数的语法
除了预设关键字之外，还设定一组规则，语法为`<前缀><动作><参数>`：
- 前缀是网站名，如npm、github、mdn等。
- 动作是单个符号，`/`代表搜索，`:`代表直接导航，在部分前缀中他们的表现是一样的。
- 参数就是参数嘛，附加到url。
- 例如 `npm:express` 就直接跳转到该包的npm页面。

目前支持的前缀：

|前缀|描述|
|:---:|:----|
|`npm`|npm搜索，可直接跳转
|`github`|`gh`，github搜索，可直接跳转
|`mdn`|用MDN搜索
|`node`| nodejs API 文档
|`next`| next.js API 文档
|`fy`|用搜狗翻译输入参数
|`g`|用谷歌搜索
