今天我们来玩网格, 现在创建了一个6列的网格。

<Demo grid={1} />

- `display: grid` 定义一个网格容器。
- `grid-template-columns(rows): <...长度>` 控制列数和各列的宽度，
写多少个就有多少列。
- `grid-[row|column]-gap` 设置格子的间隙（不影响与父容器的边距）。

这很简单，对吧？

## 定位与跨列
默认情况就是按原始顺序排列，但是可以单独为一个格子设定位置，
就像这样：

```css
.item4 {
  grid-column-start: 6;
  grid-row-start: 2;
}
```

<Demo grid={2} />

可以配合 `end` 设置跨列：
```css
.item8 {
  grid-column-start: 1;
  grid-column-end: 2;
  grid-row-start: 1;
  grid-row-end: 3;
}

.item9 {
  grid-column-start: 3;
  grid-column-end: 5;
  grid-row-start: 2;
  grid-row-end: 3;
}
```

<Demo grid={3} />

也可以用简洁的写法，看做一个图形框选系统就行了。

```css
 grid-column: 1 / 2 ;
 grid-row: 1 / 3;
 /* 或者 */
 grid-area: 1 / 2 / 1 / 3;
```

## 分数单位与repeat函数
 
 - `fr` 单位可以使单元格自适应容器尺寸。如5列的网格中，`1fr` 就是容器大小的 1/5 并减去 gap 。

- `repeat(n, expr)`  要定义10列网格，可以写10个长度值，也可以 `repeat(10, 1fr)`。
